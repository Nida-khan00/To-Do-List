# To-Do-List
JavaScript Todo App with Local Storage


## Project Description

This is a simple and interactive **Todo Application** built using **HTML, CSS, and JavaScript**.
The application allows users to manage their daily tasks efficiently by adding, completing, and deleting tasks. The tasks are also stored in **LocalStorage**, which means they remain saved even after refreshing the page.

---

## Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Store tasks in LocalStorage
* Tasks remain saved after page refresh

---

## Technologies Used

* HTML
* CSS
* JavaScript
* LocalStorage API

---

## How It Works

1. The user enters a task in the input field.
2. Clicking the **Add** button adds the task to the list.
3. Each task has two buttons:

   * **Complete** → marks the task as completed
   * **Delete** → removes the task from the list
4. All tasks are stored in **LocalStorage**, so they remain visible after refreshing the page.
